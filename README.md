# Portfolio App

A personal portfolio web app built with React. It showcases the developer's profile, skills, projects, resume, and contact details in a modern, responsive layout.

## Features

- Responsive hero section with profile image
- Fixed top navigation with mobile hamburger menu
- Light/dark theme toggle
- Project cards and skills overview
- Resume section with a downloadable PDF link
- Contact form and footer

## Technologies

- React
- JavaScript
- CSS
- Create React App

## Getting Started

1. Install dependencies

```bash
npm install
```

2. Start the development server

```bash
npm start
```

3. Open the app in your browser

- `http://localhost:3000` or the port shown by the CLI

## Project Structure

- `src/App.js` — main app layout
- `src/components/` — page sections and UI components
- `src/index.css` — global styling and responsive layout
- `public/` — static files like profile image and resume PDF

## Notes

- The resume section includes the full profile and a link to download the resume PDF.
- The theme toggle changes the site between dark and light mode.
- The navigation collapses into a mobile menu on smaller screens.
